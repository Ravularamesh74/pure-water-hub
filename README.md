# Shiva Ganga Water Supply

A professional enterprise web platform for Shiva Ganga Water Supply, Hyderabad.

## Project Overview
This application is a comprehensive business solution for managing mineral water supply operations, including a customer portal, admin dashboard, and full-featured business website.

## Features
- **Dynamic Business Website**: About, Products, Services, Quality Control, and Contact pages.
- **Customer Portal**: Order tracking, subscription management, and profile dashboard.
- **Admin Panel**: Manage orders, customers, and deliveries.
- **Modern UI**: Built with React, Tailwind CSS, and shadcn/ui with a premium water-themed aesthetic.

## Tech Stack
- **Framework**: Vite + React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui & Lucide Icons
- **State Management**: React Query

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
```sh
# Clone the repository
git clone <repository-url>

# Navigate to the project directory
cd pure-water-hub

# Install dependencies
npm install

# Start the development server
npm run dev
```
